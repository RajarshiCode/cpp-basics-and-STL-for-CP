### Prerequisites

- Install vs code.

- Install gcc/g++ compiler ie, *`MINGW-64`*

### Skeleton Code or The Basic Code for CP

```c++
#include<bits/stdc++.h>

int main()
{
    cout<<endl;
    return 0;
}
```
### If by chance if you are getting any error like
    üí° Edit "includePath" setting or,
    üí° Enable all error squiggles or,
    üí° Disable error squiggles

### Then open c_cpp_properties.json from .vscode folder and paste the code. Save and run again

```json
{
    "configurations": [
        {
            "name": "Win32",
            "includePath": [
                "C:\\MinGW\\include"
            ],
            "defines": [
                "_DEBUG",
                "UNICODE",
                "_UNICODE"
            ],
            "windowsSdkVersion": "10.0.19041.0",
            "compilerPath": "C:/MinGW/bin/g++.exe",
            "cStandard": "c17",
            "cppStandard": "c++17",
            "intelliSenseMode": "${default}"
        }
    ],
    "version": 4
}
```
